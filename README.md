Web Application Attack and Defense Lab

1. Overview

This repository contains a web application security project completed as part of a cybersecurity internship program. The project focused on identifying common web application vulnerabilities, implementing defensive security controls, and evaluating the security posture of a PrestaShop application in a controlled laboratory environment.

2. Objectives

1. Deploy and configure DVWA for security testing.
2. Perform SQL Injection (SQLi) attack simulations.
3. Perform Cross-Site Scripting (XSS) attack simulations.
4. Install and configure ModSecurity Web Application Firewall (WAF).
5. Validate that ModSecurity detects and blocks malicious requests.
6. Conduct a security assessment of a PrestaShop application.
7. Complete a web application security checklist.
8. Document findings and recommend security improvements.

3. Laboratory Environment

* Operating System: Kali Linux
* Web Server: Apache2
* Vulnerable Application: Damn Vulnerable Web Application (DVWA)
* E-commerce Platform: PrestaShop
* Web Application Firewall: ModSecurity
